# リモートデスクトップ接続ツール


## 使い方
1. RemoteDesktop_Connection.batを、右クリック
1. 編集をクリック
1. 以下の内容を書き換える
    ```
    Set SERVER=IPアドレス
    Set USERNAME="****"
    Set PASSWORD="****"
    ```
1. 書き換えた後、.batを実行する
